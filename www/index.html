The Integrative network Graphic engine Interface for R

The Integrative network Graphic engine Interface for R (IGIR) aims at bridging existing powerful graphic engines such as Cytoscape and R together to produce gene networks automatically. IGIR will provide users with a very flexible interface to take advantages of graph packages in R to do statistical analysis and visualization power of various existing graphic software.

Features

Eventually, IGIR package will have following features:

  • IGIR is programmed in object-oriented style based on R.oo package in R. So it can be easily inherited to fit users' specific requirements.
  • IGIR adopts a standard XGMML(eXtensible Graph Markup and Modeling Language) format to interact among existing network software.
  • IGIR will offer more automatic solutions to visualize gene networks.
  • IGIR will include a series of automatic pipelines for some popular bioinformatic workflows.
